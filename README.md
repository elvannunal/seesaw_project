# Seesaw Simulation

## Proje Hakkında
Bu proje, **pure JavaScript, HTML ve CSS** kullanılarak geliştirilmiş bir seesaw (tahterevalli) simülasyonudur. Kullanıcı her tıklamada farklı ağırlıklarda objeler ekler ve seesaw fizik kurallarına göre eğilir. Projenin amacı, kullanıcı etkileşimi, animasyon ve mantıksal düşünme becerilerini göstermek ve clean code pratiği yapmaktır.

## Özellikler
- Kullanıcı tıklamasıyla farklı ağırlıklarda objeler eklenir.
- Seesaw, ağırlık merkezine göre gerçekçi olarak eğilir (torque hesaplaması).
- Sol ve sağ tarafta toplam ağırlık göstergesi.
- Seesaw eğim açısı gösterimi (° cinsinden).
- "Reset" butonu ile tüm seesaw verilerini sıfırlama.
- Objeler renkli ve ağırlık bilgisiyle gösterilir.
- Log alanında atılan objelerin bilgileri listelenir.
- Responsive ve görsel olarak akıcı animasyonlar.

## Teknolojiler
- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage ile veri kaydetme/geri yükleme
